# Downloaded Papers

## Core Papers (LLM Forecasting + Prediction Markets)

1. **Leveraging Log Probabilities in Language Models to Forecast Future Events** (papers/2501.04880_log_probabilities_forecast_future.pdf)
   - Authors: Multiple (ICSC 2025)
   - Year: 2025
   - arXiv: 2501.04880
   - Why relevant: Generates novel future event forecasts with calibrated probabilities using log-probs; directly applicable pipeline architecture

2. **LLMs Can Teach Themselves to Better Predict the Future** (papers/2502.05253_outcome_driven_finetuning_forecasting.pdf)
   - Authors: Multiple
   - Year: 2025
   - arXiv: 2502.05253
   - Why relevant: Outcome-driven fine-tuning using Polymarket data; demonstrates prediction markets as LLM training signal

3. **Forecasting Future Language: Context Design for Mention Markets** (papers/2602.21229_mention_markets_context_design.pdf)
   - Authors: Multiple
   - Year: 2026
   - arXiv: 2602.21229
   - Why relevant: Market-Conditioned Prompting shows LLMs + markets are complementary; MixMCP achieves best results

4. **Time-R1: Towards Comprehensive Temporal Reasoning** (papers/2505.13508_time_r1_temporal_reasoning.pdf)
   - Authors: UIUC
   - Year: 2025
   - arXiv: 2505.13508
   - Why relevant: Generates future news headlines with temporal grounding; 3B model matches 671B; provides AvgMaxSim evaluation metric
